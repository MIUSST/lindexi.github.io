
# win10 uwp 在 VisualStudio 部署失败，找不到 Windows Phone 可能的原因

在我使用 VisualStudio 调试的时候，发现我插入了手机，但是 VisualStudio 在部署的时候找不到手机。
可能的原因是 手机禁用了连接，第二个原因是可能手机驱动没正确让 VisualStudio 找到手机
